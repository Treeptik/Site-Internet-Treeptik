CloudUnit - le PaaS Java

 

Déployez vos applications Java EE sur le Cloud

Treeptik spécialiste des applications Java JEE et du cloud computing développe une plateforme innovante de type PaaS baptisé CloudUnit.


 


C

loudUnit est un projet issu du pôle recherche et développement de Treeptik et consiste en la mise en oeuvre d'une offre de services Cloud PaaS destinée aux équipes de développement Java. Elle a pour but de réduire considérablement les temps de développement et de mise en production des applications web développées par ces mêmes équipes.

L

' offre de service externalisée, CloudUnit, permet aux développeurs de s'affranchir de la création des serveurs, des bases de données et de tous les outils qui constituent l’environnement de l'application et qui sont nécessaire à son bon fonctionnement. CloudUnit prend en charge la création, la configuration et le suivis opérationnel de tous ces outils qui sont soit déployé sur un cloud public (chez nous) soit sur un cloud privé (chez nos client). Les développeur on juste besoin de pousser le code source de l'application et la plateforme se charge de la compiler et de démarrer tous les outils nécessaire.

C

ette offre apporte une réelle avancé dans l'industrialisation du développement logiciel et permet à toute les équipes de se concentrer sur leur cœur de métier. Les développeur peuvent ainsi se concentrer sur le développement de nouvelles fonctionnalité et les équipes d'infrastructures sur le maintien en condition opérationnel des applications de production. Ce qui représente un atout réel et un gain en compétitivité important.

Fonctionnalités

  • Aucunes modifications dans les applications
  • Déployez vos applications avec une commande Git
  • Déployoiement avec une interface Web ou en ligne de commande
  • Release automatique
  • Rollback simplifié
  • Paiement à l’utilisation

Les avantages


  • Coût maitrisé
  • Plus de sécurité
  • Un SI plus agile
  • Plus de productivité
  • Un cout d'innovation réduit
