CloudUnit - le PaaS Java

 

Déployez vos applications Java EE sur le Cloud

Treeptik spécialiste des applications Java JEE et du cloud computing développe une plateforme innovante de type PaaS baptisée CloudUnit.


 


C

loudUnit est un projet issu du pôle recherche et développement de Treeptik. C'est une offre de services de type Cloud PaaS destinée aux équipes de développement Java. Elle a pour but de réduire considérablement les temps de développement et de mise en production des applications web.

L

' offre de service externalisée, CloudUnit, permet aux développeurs de s'affranchir de la création des serveurs, des bases de données et de tous les outils qui constituent l’environnement de l'application et qui sont nécessaires à son bon fonctionnement. CloudUnit prend en charge la création, la configuration et le suivis opérationnel de tous ces outils qu'ils soient déployés sur un cloud public (chez nous) ou sur un cloud privé (chez nos client). Les développeurs n'ont qu'à pousser le code source de l'application et la plateforme se charge de la compiler et de démarrer tous les outils nécessaires.

C

loudUnit apporte une réelle avancée dans l'industrialisation du développement logiciel et permet à toutes les équipes de se concentrer sur leur cœur de métier. Les développeurs peuvent ainsi se concentrer sur le développement de nouvelles fonctionnalités et les équipes d'infrastructures sur le maintien en condition opérationnel des applications de production. Ce qui représente un atout réel et un gain en compétitivité important.

Fonctionnalités

  • Aucunes modifications dans les applications
  • Déployez vos applications avec une commande Git
  • Déploiement avec une interface Web ou en ligne de commande
  • Release automatique
  • Rollback simplifié
  • Paiement à l’utilisation

Les avantages


  • Coûts maîtrisés
  • Plus de sécurité
  • Un SI plus agile
  • Meilleure productivité
  • Un coût d'innovation réduit
